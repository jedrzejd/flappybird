## Running the application

```bash
$ git clone https://github.com/jedrzejd/flappybird.git
$ cd flappybird
$ python3 -m venv venv
$ . venv/bin/activate
$ pip install -r requirements.txt
$ python flappy_bird.py
```
To exit venv type in the console:
```
$ deactivate
```
